# NutriSmart

## Projekt leírás
A NutriSmart egy webes alkalmazás, amely a felhasználó által megadott egészségügyi és életmódbeli adatok alapján személyre szabott étrendet állít össze.  
Az alkalmazás figyelembe veszi az allergiákat, betegségeket és a felhasználó célját (pl. fogyás, izomépítés).

## Célközönség
Az alkalmazás célközönsége mindazok, akik tudatosan szeretnének odafigyelni az egészségükre és táplálkozásukra.

## Technológiák
- Backend: Java 17, Spring Boot
- Frontend: React, HTML5, CSS3, JavaScript
- Adatbázis: MySQL
- Verziókezelés: GitHub
